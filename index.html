HTML Introduction

What is HTML?

  • Standar for Hyper text Markup Lenguage
  • Describes the structure of a Web Pages
  • Consist of a series of elements
  • Elements tell the browser how to display the content
  • Elements consist on pieces label with content

Basics Elements Explained

  1. La etiqueta "DOCTYPE" define la declaracion del documentoen cuestion como HTML
  2. La etiqueta HTML es el elemento principal de una pagina HTMLL
  3. La etiqueta "head" contiene información adicional acerca de la pagina HTML
  4. La etiqueta "title" especifica el titulo de una pagina web
  5. La etiqueta "body" especifica el cuerpo de una pagina web(todos los elementos visibles)
  6. La etiqueta "h1" define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño subtitular
  8. La etiqueta "p" define un texto de tamaño parrafo

Que es un elemento HTML?

Un elemento HTML esta definido por una etiqueta de inicio seguido de algun contenido y una etiqueta de finalización
  • <nombreEtiqueta>: equiqueta de inicio
  • Contenido:entre la etiqueta de inicio y la etiqueta de finalización
  • </nombreEtiqueta>: etiqueta de finalización

Etiqueta <map>

La etiquetaueta <map> define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distintas areas detectables
por el navegador, logrando identificar de manera independiente cada una de las areas
La etiqueta <map> define las areas de la imagen mediante la etiqueta <area> con los atributos HTML "shape" que define
la forma del area señalada en la imagen, "coords" que define las coordenadas de cada uno de los puntos de shape, "href" que puede definir
un hipervínculo para cada una de las areas.
Ejemplo:
<img src="workflow.jpg" usemap="#workmap">
<map name="workmap">
< area shape = "rect" coord="34,44,270,350" href="pagina1.com">
< area shape="circle" coord = "200,172,33,250" href="pagina2.com"
< /map>

Elemento Table en HTML

La etiqueta <table> permite colocar los datos en filas y columnas usando sus derivaciones <tr> para definir una fila(table row),<th>
para definir los encabezados de la tabla (table header) y <td> para asignar valores a las filas correspondientes a los encabezados de las columnas(table data)
Ejemplo:
<table>
<tr>
<th>Nombre</th>
<th>Apellido</th>
<th>Edad</th>
<th>Dirección</th>
</tr>
<tr>
<td>Eduardo</td>
<td>Campos</td>
<td>18</td>
<td>Carre 38 #3 Col.Arboledas Qro.</td>
</tr>
</table>
